## 前提

- Dockerfile にあるとおりの Python バージョン

## 作りたいスクリプトの仕様

main_001_discord_to_jsonl.py

1. コマンドライン引数として以下をとる
    - `--discord-application-id`: 必須
    - `--discord-public-key`: 必須
    - `--discord-bot-token`: 必須
    - `--discord-channel-id`: 必須
    - `--discord-user-id`: 任意 (デフォルト: None)
    - `--output`: 任意 (デフォルト: `main_001_discord_to_jsonl.jsonl`)
    - `--date-from`: 任意 (デフォルト: None)
    - `--limit`: 任意 (デフォルト: None)
2. discord.py (インストール済み) を利用して、チャンネルからメッセージを取得する
3. 取得したメッセージを jsonl 形式で保存する
    - ディレクトリ: output/{引数で指定されたもの}

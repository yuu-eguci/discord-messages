## 前提

- Dockerfile にあるとおりの Python バージョン
- `main_002_rawjsonl_to_parsedjsonl.jsonl` にあるような形式の jsonl を扱う

## 作りたいスクリプトの仕様

main_003_parsedjsonl_to_anotherformat.py

1. コマンドライン引数として以下をとる
    - `--input`: 必須
    - `--format`: 任意 (デフォルト: "anki"; 選択肢: "anki", "fancyhtml")
    - `--output`: 任意 (デフォルト: format に応じて変わる。)
2. fancyhtml
    - jsonl から heading と children の組み合わせを取得 (形式は実際のファイルを参照)
    - heading をクリックすると children がトグルで出てくる、というのが縦に並んでいるような html (css を含む) を出力する
    - `main_003_parsedjsonl_to_anotherformat_fancyhtml.html` というファイル名で出力する
    - この処理は formatter モジュールを作って切り出す
3. anki
    - 下の anki format を使って jsonl を処理
    - `main_003_parsedjsonl_to_anotherformat_anki.txt` というファイル名で出力する

***

- 使ってみて気づいた点を修正: main_002_rawjsonl_to_parsedjsonl.jsonl の段階で空っぽの場合がある `[]` この場合は html や anki txt への変換時にスキップする。

これを修正お願い。

### anki format

こういう場合……

```json
[{"heading": "foo", "children": ["bar", "baz"]}, {"heading": "qux", "children": ["quux"]}]
```

こんなふうに。ただし `<TAB>` のところは本当にタブ文字を出力する。

```plaintext
#separator:tab
#html:true
foo<TAB>bar<br />baz<TAB>discord
qux<TAB>quux<TAB>discord
```
